A — Per-field edit icons

Pencil icon ✎ on each row. Clicking it triggers the AI to ask specifically for that field. Precise and quick — no need to say which field to change. Summary re-renders once new value is validated.

B — "Edit" button → AI asks which field
Confirmation card in chat:

B — "Edit" button + field chips

Single "Edit" button. Clicking it has the AI ask "Which field?" and shows quick-reply chips for each field. More conversational. User can also just type "change my email" in natural language.

In both cases, after the new value is validated, the summary card refreshes with the updated detail. Click "A" or "B" or tell me in the terminal.
